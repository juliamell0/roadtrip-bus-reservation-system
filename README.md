# 🚍 RoadTrip Bus Reservation System

Sistema de **gestão de reservas de passagens de ônibus** desenvolvido em **Java**, com menu interativo no terminal.

---

## 📌 Sobre o projeto

Este projeto simula um sistema utilizado por atendentes de uma empresa de transporte para **gerenciar reservas de passagens de ônibus**.

O programa permite cadastrar ônibus, registrar assentos disponíveis e realizar reservas para passageiros.

---

## ⚙️ Funcionalidades

✔ Registrar até **4 ônibus**
✔ Registrar **quantidade de assentos disponíveis**
✔ Realizar **reservas de passagens**
✔ Consultar reservas por **ônibus**
✔ Consultar reservas por **passageiro**
✔ Sistema com **menu interativo no terminal**

---

## 📋 Regras do sistema

* Máximo de **4 ônibus cadastrados**
* Máximo de **20 reservas**
* Verificação se o **ônibus existe**
* Verificação de **assentos disponíveis**
* Mensagens de erro quando necessário
